# Projektidee
Wir untersuchen wie sich die maternalen Sterblichkeiten von 1985 bis 2023 entwickelt hat. Wir beziehen uns dabei auf die 10 Ländern, mit den meisten Todesfällen. Zudem recherchieren wir nach möglichen Gründen dafür.
